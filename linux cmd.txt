Insert data in  A FILE:
-----------------------------------

1) cat: (concatenation) 
	- to used show the content of a file
	- to create static data in a file
	Eg:
		cat > filename
		this is first data
		jlksjdflkjljl
		ctrl + d
		
		(or)
		cat >> filename
		this is first data
		jlksjdflkjljl
		ctrl + d


	> 	--> override
	>> 	--> append

	concatenation: cat file1 file2 >> file3

	tac : used to show data in reverse order 

2) touch : (timestamp)
	- used to create empty file or multiple files
	eg:
		touch java{1..10}
		touch {1..2}.java
	
3) echo
	- used to create single line data in file
	eg:
		echo " some string " > file_name		--> used to override
				(or)
		echo "some string " >> file_name		--> used to append

4) vi / vim: 
	- used to edit the data or view the data of a file
	
	vi file_name

	--> press 'i' in k/b
	--> enter data
	--> esc --> :wq

	--> w - save, q - quit , ! - forcebly

	Note: 
		- vi is always faster nano
5) nano
	- used to edit the data or view the data of a file
	
	nano file_name

	--> enter data 
	--> ctrl + x  >> shift + y >> enter
